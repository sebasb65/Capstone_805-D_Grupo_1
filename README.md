Tecnologias necesarias para el proyecto:

Node.js: Para instalar librerias.
Ionic (Angular): Nuestro Frimework.


Paso a paso:

Paso 1: Clonar nuestro repositorio en tu equipo.
Paso 2: Abrir nuestro proyecto con Visual Studio Code.
Paso 3: Abrir la terminal de Visual Studio Code de preferencia con Cmd.
Paso 4: Escribir 'cd YouConApp_Ionic_Angular_Firebase', para ir a la carpeta del proyecto.
Paso 5: Una vez este en esa ruta debes ingresar el comando 'npm install', esto se encargara de instalar todas las librerias necesarias para que el proyecto funcione.
Paso 6: Una vez que se descarguen solo queda escribir el comando 'ionic serve', con esto solo queda probar la aplicacion :)
